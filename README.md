# 📚 Multi-Paper Comparison RAG System

## 🎯 What is this?
An advanced RAG (Retrieval-Augmented Generation) system that allows you to **compare multiple research papers simultaneously**. Unlike simple PDF chatbots that handle one document, this system can analyze and contrast multiple papers side-by-side.

## ✨ Key Features
- **Multi-paper comparison** - Compare methodologies, results, and contributions
- **Semantic search** - Find relevant passages across all papers
- **Paper statistics** - Analyze document structure and content
- **Common theme detection** - Discover shared topics automatically
- **Visual analytics** - Interactive charts and metrics

## 🛠️ Technical Implementation
- **Vector Database**: ChromaDB for efficient similarity search
- **Embeddings**: Sentence Transformers (all-MiniLM-L6-v2)
- **UI Framework**: Streamlit for interactive dashboard
- **PDF Processing**: PyPDF2 for text extraction
- **Analytics**: Pandas + Plotly for data visualization
